What we did

  • Installed, or had a hard time installing, Armadillo, a C++ linear algebra library.

Summary

Install armadillo. Download the source, and follow the instructions. On linux with dependencies (cmake etc.) already installed: and if everything works fine,

One might confront various issues. Notably, g++ (or other compilers) might not search for the directory where armadillo is installed by default. You can tell g++ to search for a directory.

Another option you might have to specify is -DARMA_DONT_USE_WRAPPER. Possibly like this: 


      g++ -I/usr/local/include example1.cpp -o example1 -O2 -I /home/blah/armadillo-5.100.2/include -DARMA_DONT_USE_WRAPPER -lblas -llapack
